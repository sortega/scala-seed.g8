# Simple scala project template

This is a [Giter8][g8] template for Scala

```bash
$ sbt new sortega/scala-seed.g8
```

## How to test

```bash
$ sbt test
```

Instantiate it locally:

```bash
$ cd /tmp
$ g8 file://path/to/here/scala-seed.g8/ --name=testproject --force
```
